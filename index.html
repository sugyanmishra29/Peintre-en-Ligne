How to use?

  • New: Clicking this will open a new tab with a new canvas

  • Erase: Clicking this will wipe out all the content in the canvas

  • Download: Use this feature to save your painting locally as a PNG file

  • Pen: Say you draw a line. It'll then have circular ends

  • Brush: Say you draw a line. It'll then have rectangular ends

  • Paintbrush Size: Enter paintbrush size (in pixels) in textfield. Do not press ENTER key after input

  • Color Picker: Choose color of pen/brush using the color picker
